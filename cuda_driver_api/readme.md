# CUDA Driver API Documentation

CUDA Driver API documentation in LLM-readable markdown format.

## Structure

- `texts/cuda_docs/sections/` - Individual section markdown files
- `texts/cuda_docs/cuda_driver_api_full.md` - Complete combined documentation
- `texts/cuda_docs/_index.txt` - Index of all sections

## Usage

To regenerate documentation:

```bash
uv run scrape.py
```
